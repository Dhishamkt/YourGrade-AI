YourGrade AI 🎓
An AI-Powered Student Performance Analyser built with React, Flask, Pandas, and Groq AI.

Features
    User Registration & Login (localStorage)
    Upload CSV or Excel marks sheet
    AI-powered performance analysis
    Personalised study recommendations
    Best & weak subject detection
    Clean, responsive UI with blue theme
    Tech Stack
Frontend:
    React + Vite
    React Router DOM
    Tailwind CSS
Backend:
    Flask (Python)
    Flask-CORS
    Pandas
    Groq AI API (LLaMA 3.3 70B)

⚙️ Installation & Setup
Frontend
    npm install
    npm run dev

Backend
    cd backend
    pip install -r requirements.txt
    python app.py

Root .env
    VITE_GROQ_API_KEY=your_key_here

backend/.env
    GROQ_API_KEY=your_groq_key_here
